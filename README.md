# suipta

Simple boilerplate-code cli generator intended to imporve your experience with Feature-Sliced Design methodology

## Contributing

### Requirments:

- volta

### Install tooling:

```sh
volta install node pnpm lefthook commitizen
```

### Install dependencies:

```sh
pnpm i
```

### Run tests:

In watch mode

```sh
pnpm test
```

One time

```sh
pnpm test:run
```

### Lint code:

```sh
pnpm lint
```

### Format code:

```sh
pnpm format
```
